# Plugin Assets for WordPress.org

This directory contains visual assets for the WordPress.org plugin listing.

## Structure

- **screenshot-1.png** - Password reset form with CAPTCHA protection
- **screenshot-2.png** - Admin settings page with difficulty selector
- **screenshot-3.png** - Mobile responsive view
- **banner-772x250.png** - Standard banner for plugin listing
- **banner-1544x500.png** - High-DPI/Retina banner
- **icon-128x128.png** - Standard plugin icon
- **icon-256x256.png** - Retina plugin icon
- **icon.svg** (optional) - Vector plugin icon

## Requirements

- Maximum 10MB per screenshot
- Maximum 4MB per banner
- Maximum 1MB per icon
- PNG, JPG, and GIF formats supported
- Banners should be 772x250px (standard) or 1544x500px (Retina)
- Icons should be square images

## Notes

All visual assets are optional but recommended for a professional WordPress.org listing. Screenshots and banners help users understand what your plugin does before installation.
