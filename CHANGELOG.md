# Changelog

All notable changes to Password Reset Guard will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-07

### Added
- Initial release of Password Reset Guard
- Native math CAPTCHA (addition, subtraction, multiplication)
- Three difficulty levels (Easy, Medium, Hard)
- Admin settings page in WordPress Settings
- CAPTCHA field injection into password reset form
- Server-side CAPTCHA validation
- Client-side validation feedback
- Lightweight CSS styling with responsive design
- JavaScript form handling with accessibility support
- Full internationalization (i18n) support
- Translation template file (POT)
- Comprehensive documentation and README
- Security best practices implementation
- Support for WordPress 4.9+

### Features
- Zero dependencies or external services
- Privacy-focused (no third-party CAPTCHA services)
- Lightweight footprint (minimal PHP and JavaScript)
- Mobile-friendly responsive design
- Keyboard navigation support
- Screen reader compatible
- High contrast mode support
- Easy configuration in WordPress admin panel
